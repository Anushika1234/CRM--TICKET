# CRM Ticket Management System

A web-based CRM Ticket Management System built using:

- PHP
- MySQL
- Bootstrap 5
- HTML, CSS, JavaScript

## Features
- User Authentication (Login / Register)
- Ticket Creation & Assignment
- Role-based Access (Author / Assignee)
- Status Management
- Secure Session Handling

## How to Run
1. Install XAMPP
2. Place project in `htdocs`
3. Import database
4. Start Apache & MySQL
5. Open in browser
